Get started

The Busly API provides programmatic access to read Busly Game data.

To use this Iframe, you need a Client key. Please contact us at techsupport@shacksevo.co to get your own API key.

Game Api

Game is accessed via iframe src=https://busly-service.centralus.azurecontainer.io/.


QUERY PARAMETERS

Field Type Description
client_key String Your API key.
userid String New unique userid.
client api String Url for api to verify user and update bet status.

Verify User

Provide an api to authorize user :
/verify_user returns true if user is allowed to bet and vice versa


PARAMETERS

Field Type Description
userid String Unique user.
client seed String Generated client seed.
Game Round ID String New game round ID.
Stake Number Bet stake.
