Get started

The Busly API provides programmatic access to read Busly Game data.

To use this API, you need an API key. Please contact us at management@shackeevo.com to get your own API key.

Register New User

To register a new user you need to make a POST call to the following url :
register/


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
uname String New unique username.
pw String A secure password.

Login

To login a user you need to make a POST call to the following url :
login/


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
username String A registered username.
password String A registered user's password.

Auto Create User

To create a user you need to make a GET call to the following url :
/api/createUser


Note: It automatically logs user in.

>

Validate User

To validate a user you need to make a POST call to the following url :

/api/validate_user


Validate userid.


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
userid Number A registered user's ID.

Get Past Odds

To get past odds you need to make a GET call to the following url :
/api/getPastOdds


Returns list of Past Game Odds.


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.

Get Jackpot

To get current Jackpot you need to make a GET call to the following url :
/api/getJackpot


Returns current Jackpot.


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.

Place Bet

To place a bet you need to make a POST call to the following url :
/api/place_bet


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
userid String User's ID.
amount_staked Number Amount to stake.
client_seed String client seed
game_round_ID Number game round ID.

Events Place Bet

To place an event type bet you need to make a POST call to the following url :
/api/EVENTS_place_bet


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
userid String User's ID.
amount_staked Number Amount to stake.
client_seed String client seed
game_round_ID Number game round ID.
selected_event_type Number { 0: 'Overheating', 1: 'Accident' , 2: 'Flat Tyre' , 3: 'Traffic Officer', 4: 'Bad Road', }
selected_range_type Number Selected Range Type.
potential_winnings Number Potential Winnings.

Delete Bet

To delete a bet you need to make a POST call to the following url :
/api/delete_bet


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
userid String User's ID.

Update Bet

To update a bet you need to make a POST call to the following url :
/api/delete_bet


QUERY PARAMETERS

Field Type Description
secret_key String Your API key.
RUNNING_bet_waiting_id Number running bet ID.

Errors

The Westeros API uses the following error codes:

Error Code Meaning
X000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
X001 Unknown or unvalid secret_key. This error appears if you use an unknow API key or if your API key expired.
X002 Unvalid secret_key for this domain. This error appears if you use an API key non specified for your domain. Developper or Universal API keys doesn't have domain checker.
X003 Unknown or unvalid user token. This error appears if you use an unknow user token or if the user token expired.
