Get started

The Busly API provides programmatic access to integrate with Busly Game.

To use this API, you need a Client key. Please contact us at techsupport@shacksevo.co to get your own API key.

Game Access

Game can be accessed via URL "'GAME_URL'+'&user_name='+CURRENT_USER_NAME+'&api='+YOUR_API_ENDPOINT" Or iframe :src="'GAME_URL'+'&user_name='+CURRENT_USER_NAME+'&api='+YOUR_API_ENDPOINT"

QUERY PARAMETERS

Field Type Description
CURRENT_USER_ID String New unique userid.
YOUR_API_ENDPOINT String Url for api to verify user and update bet status.

Verify User

Provide an api endpoint to authorize user :
POST {{your_api_endpoint}}/verifyuser

return the value "canBet" as true if user is allowed to bet and false if otherwise.


PARAMETERS

Field Type Description
username String Unique user.
client_seed String Generated client seed.
game_round_ID String New game round ID.
stake_amount Number Bet stake.

RESPONSE

Field Type Description
canBet Boolean return the value "canBet" as true if user is allowed to bet and false if otherwise.
message String success message/ deciptive error message

Update User Bet

Provide an api endpoint to update user bet on completion :
POST {{your_api_endpoint}}/update_bet
